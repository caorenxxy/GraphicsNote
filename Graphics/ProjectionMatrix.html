投影矩阵

推到从camera space(Ωcam\Omega_{cam}Ωcam​)到NDC space(ΩNDC\Omega_{NDC}ΩNDC​)的投影矩阵，这里NDC space的x,y,zx,y,zx,y,z的范围都是[−1,1][-1,1][−1,1]。假设在camera space里的坐标为 (x,y,x)(x,y,x)(x,y,x)，我们把这个点投影到film上，投影得到上的坐标为(xp,yp,zp)(x_p,y_p,z_p)(xp​,yp​,zp​)，在NDC space中的坐标为(xN,yN,zN)，(x_N, y_N,z_N)，(xN​,yN​,zN​)，film距离camera的距离为filmDisfilmDisfilmDis，根据fovyfovyfovy以及filmDisfilmDisfilmDis可以计算出film在camera space里的横坐标范围为[l,r][l,r][l,r]，纵坐标范围是[b,t][b,t][b,t]，定义投影矩阵Mproj:Ωcam→ΩNDCM_{proj}: \Omega_{cam}\to \Omega_{NDC}Mproj​:Ωcam​→ΩNDC​。并且这里假设近裁剪端和远裁剪端到camera的距离分别为n,fn,fn,f(这里需要了解齐次坐标的知识)

由几何关系
xpx=ypy=filmDisz\frac{x_p}{x} = \frac{y_p}{y} = \frac{filmDis}{z}xxp​​=yyp​​=zfilmDis​
可以得到
xp=filmDisz⋅xyp=filmDisz⋅y\begin{gathered} x_p = \frac{filmDis}{z} \cdot x\\ y_p = \frac{filmDis}{z} \cdot y \end{gathered}xp​=zfilmDis​⋅xyp​=zfilmDis​⋅y​
对于xpx_pxp​我们可以得到以下推导
l≤xp≤r0≤xp−l≤r−l0≤xp−lr−l≤10≤2xp−lr−l≤2−1≤2xpr−l−r+lr−l≤1\begin{gathered} l \leq x_p \leq r\\ 0 \leq x_p - l \leq r-l\\ 0 \leq \frac{x_p - l}{r - l} \leq 1\\ 0 \leq 2\frac{x_p - l}{r - l} \leq 2\\ -1 \leq \frac{2 x_p}{r-l} - \frac{r + l}{r - l} \leq 1 \end{gathered}l≤xp​≤r0≤xp​−l≤r−l0≤r−lxp​−l​≤10≤2r−lxp​−l​≤2−1≤r−l2xp​​−r−lr+l​≤1​
同理也可以得到
−1≤2ypt−b−t+bt−b≤1-1 \leq \frac{2y_p}{t - b} - \frac{t + b}{t - b} \leq 1−1≤t−b2yp​​−t−bt+b​≤1
总结一下，我们可以的得到关于x,yx, yx,y的投影后的关系
{−1≤2r−l⋅filmDisz⋅x−r+lr−l≤1−1≤2t−b⋅filmDisz⋅y−t+bt−b≤1\left\{ \begin{gathered} -1 \leq \frac{2}{r - l}\cdot\frac{filmDis}{z}\cdot x - \frac{r + l}{r - l} \leq 1\\ -1 \leq \frac{2}{t - b}\cdot\frac{filmDis}{z}\cdot y - \frac{t + b}{t - b} \leq 1 \end{gathered} \right.⎩⎪⎪⎪⎨⎪⎪⎪⎧​−1≤r−l2​⋅zfilmDis​⋅x−r−lr+l​≤1−1≤t−b2​⋅zfilmDis​⋅y−t−bt+b​≤1​
由此我们可以构造出一部分投影矩阵的元素
Mproj=[2filmDisr−l0−r+lr−l002filmDist−b−t+bt−b000AB0010]M_{proj} = \left[ \begin{gathered} \frac{2filmDis}{r - l} & 0 & -\frac{r + l}{r - l} & 0\\ 0 & \frac{2filmDis}{t - b} & -\frac{t + b}{t - b} & 0\\ 0 & 0 & A & B\\ 0 & 0 & 1 & 0 \end{gathered} \right]Mproj​=⎣⎢⎢⎢⎢⎢⎢⎢⎢⎡​r−l2filmDis​000​0t−b2filmDis​00​−r−lr+l​−t−bt+b​A1​00B0​⎦⎥⎥⎥⎥⎥⎥⎥⎥⎤​

[xN/wNyN/wNzN/wN1]=[2xr−l⋅1z−r+lr−l2yt−b⋅1z−t+bt−b(zA+B)/z1]\left[\begin{gathered} x_N/w_N\\ y_N/w_N\\ z_N/w_N\\ 1 \end{gathered}\right] = \left[\begin{gathered} \frac{2x}{r - l}\cdot\frac{1}{z} - \frac{r + l}{r - l}\\ \frac{2y}{t - b}\cdot\frac{1}{z} - \frac{t + b}{t - b}\\ (zA + B) / z \\ 1 \end{gathered}\right]⎣⎢⎢⎢⎢⎢⎡​xN​/wN​yN​/wN​zN​/wN​1​⎦⎥⎥⎥⎥⎥⎤​=⎣⎢⎢⎢⎢⎢⎢⎢⎢⎡​r−l2x​⋅z1​−r−lr+l​t−b2y​⋅z1​−t−bt+b​(zA+B)/z1​⎦⎥⎥⎥⎥⎥⎥⎥⎥⎤​
